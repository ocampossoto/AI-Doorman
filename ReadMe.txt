To run this you will need.
1. Raspberry pi with the lates version of raspbian and ffmpeg installed (use sudo apt-get install ffmpeg)
2. Motion sensors HC-SR501 wired to pin 4 on the raspberry pi
3. Raspberry Pi Camera connected to the raspberry pi
4. Relay for Arduino connected to pin 18 on the raspberry pi

Create a folder at "/home/pi/" called "Hackathon_Video". (mkdir Hackathon_Video" in the terminal) 
Next add the email address for to and from labled "fromaddr" and "toaddr"
Next add the pasword for the from address on the server.login(fromaddr, "pasword")
Next run the code
